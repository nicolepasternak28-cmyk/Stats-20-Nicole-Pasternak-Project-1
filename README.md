# Project 1: Basic Reproducibility and Reporting

This project was completed for UCLA Stats 20 during Spring 2026. The report demonstrates the principles of reproducible data analysis using R Markdown by combining code, written interpretation, tables, and visualizations into a single document.

## Project Overview

The analysis uses a Pokémon dataset to explore statistical summaries and graphical visualization techniques in R. All outputs are generated directly from code to ensure transparency, consistency, and reproducibility.

## Topics Covered

- Reproducible reporting with R Markdown
- Loading and inspecting datasets in R
- Scatterplot visualization
- Summary statistics
- Vectorized operations
- Image integration within reports

## Files Included

- `Project1.qmd` or `Project1.Rmd` — source report file
- `Project1.html` — rendered HTML report
- `pokemon.RData` — dataset used in analysis
- `README.md` — project description

## Tools Used

- R
- R Markdown / Quarto
- knitr
- Base R graphics

## Key Analysis

The project investigates relationships between Pokémon attributes such as attack and defense using scatterplots and descriptive statistics. Additional sections demonstrate vectorized transformations and reproducible workflows commonly used in statistical computing.
